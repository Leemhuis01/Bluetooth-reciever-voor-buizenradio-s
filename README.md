# Bluetooth-reciever-voor-buizenradio-s
Bluetooth ontvanger voor buizenradio's en versterkers

Met deze handige stekker kan bluetooth worden toegevoegd aan de meeste buizen radio's/versterkers.
Deze stekker kan door middel van 2 banaanstekkers worden aangesloten op de radio.
Vervolgens kan middels de USB-C connector de stekker worden voorzien van een spanning van 5V

LET OP!!!!
De stekker zou ook in een stopcontact passen.
Dit is niet de bedoeling.
Hiermee is de bluetooth module kapot,
en nog erger dat is heel erg gevaarlijk voor de gebruiker.
